# Python Spell checker



## Installation

```bash
# for installing the lib to use
pip install symspellpy
pip install cyhunspell

# setting up api
pip install fastapi
pip install uvicorn[standard]
```

## Usage

```python
# window
env\Scripts\activate

# mac or linux
source env/bin/activate
```
